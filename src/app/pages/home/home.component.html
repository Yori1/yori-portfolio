Kanji Shoubu

Kanji Shoubu is a webapp built with Spring Boot and MySQL. It's an application that allows logged in users to view a list of all online users, and challenge one of the users to a game of trying to read Japanese sentences. When the challenge has been accepted, both players try to correctly read the same sequence of sentences. The user guesses the reading by typing in the reading in either alphabet characters or Japanese characters, and is then shown the next sentence. At the end, the player with the largest amount of correct readings wins, if that's the same the fastest player wins. This is all accomplished using Spring Boot websocket connections combined with StompJS.
